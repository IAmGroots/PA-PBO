# Proyek Akhir PBO
**Judul PA**
- Sistem Pasar Eletronik
**Deskripsi Singkat**
- Sistem Pasar Elektronik adalah sistem yang memungkinkan pembeli dan penjual untuk berinteraksi dan melakukan transaksi.
- Ada tiga jenis pengguna dalam sistem pasar elektronik ini:
1. Admin
2. Pembeli
3. Penjual
